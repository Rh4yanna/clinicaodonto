# 🦷 Clínica Odontológica CR

## Sobre o Projeto

O Clínica Odontológica CR é um sistema desenvolvido para auxiliar na gestão de uma clínica odontológica universitária, permitindo o controle de pacientes, consultas, estoque de materiais e processos de esterilização.

A solução é composta por:

- Sistema Web para a Recepção
- Aplicativo Mobile para Professores, Alunos, CME e Almoxarifado
- API responsável pela integração entre os sistemas
- Banco de Dados SQL para armazenamento das informações

---

## Objetivo

O projeto tem como objetivo centralizar e otimizar os processos da clínica odontológica, proporcionando maior controle, rastreabilidade e organização dos atendimentos e materiais utilizados.

---

## Perfis de Usuário

### Recepção (Sistema Web)

- Cadastro de pacientes
- Agendamento de consultas
- Confirmação de consultas
- Triagem de pacientes
- Consulta de histórico de atendimentos

### Professor (Aplicativo Mobile)

- Supervisão dos atendimentos
- Aprovação de procedimentos
- Acompanhamento de cirurgias

### Aluno (Aplicativo Mobile)

- Registro de atendimentos
- Evolução clínica dos pacientes
- Registro de procedimentos e cirurgias

### CME (Aplicativo Mobile)

- Controle de esterilização
- Controle biológico
- Geração e leitura de QR Code
- Controle de pacotes esterilizados

### Almoxarifado (Aplicativo Mobile)

- Controle de estoque
- Entrada e saída de materiais
- Leitura de código de barras
- Controle de materiais por categoria
